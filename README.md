# Smart Ambulance Routing System

A smart web-based system designed to help ambulances find the fastest and most efficient routes using real-time traffic data. This project aims to reduce emergency response time and improve patient survival rates.

---

##  Features

*  Real-time traffic-aware routing
*  Interactive map for route visualization
*  Source to destination optimization
*  Fast route calculation
*  Emergency-focused navigation

---

##  Tech Stack

* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Database:** Firebase
* **Maps API:** Google Maps API
